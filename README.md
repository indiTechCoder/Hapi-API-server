# Hapi Server API with social login


## Installation

**In Development**

```sh
git clone https://github.com/kumartarun/Hapi-API-server.git
cd Hapi-API-server/
npm install
touch authCredentials.js
```

Source code can be found [here](https://github.com/kumartarun/Hapi-API-server.git).

## Usage

```bash
npm start
```
